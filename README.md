# デバッグの仕方
デバッグしたいファイルがアクティブの状態で  
「F5」  
入力したデータが存在する場合は以下のファイルに記入  
「input.txt」  
まだデバッグ実行後a.outファイルが生成されるが削除してよい  
これはビルドされたもの

# テストの仕方

テストしたいファイルがアクティブの状態で  
「ctrl + alt + z」  
実行されるとファイルがあるディレクトリにtestフォルダが作成されサンプルが用意される
